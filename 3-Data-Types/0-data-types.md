# JavaScript Data Types
## There are total 7 data types in JavaScript
Remember this word for interview. **`SNBNUSO`**

The latest ECMAScript standard defines eight data types:

 - Seven data types that are primitives:
	- String
	- Number
	- Boolean
	- Null
	- Undefined
	- BigInt
	- Symbol
 - and Object
